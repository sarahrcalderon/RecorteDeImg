Funcionalidades
# Carregar Imagem: O usuário pode carregar uma imagem clicando no botão "Carregar Imagem".

# Remover Fundo: Após carregar a imagem, o usuário pode clicar em "Remover Fundo" para remover o fundo da imagem.

# Salvar Imagem: O usuário pode salvar a imagem sem fundo clicando em "Salvar Imagem".
